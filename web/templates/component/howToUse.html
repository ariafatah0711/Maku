Ikuti langkah-langkah mudah di bawah ini

Clone project MaKu ke komputer Anda

git clone {{ REPO_URL }}.git
cd maku

(Opsional tapi sangat disarankan)

python3 -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

Install semua package yang diperlukan

pip3 install -r req.txt

Pilih salah satu interface untuk dijalankan

python3 maku.py web    # Web interface
python3 maku.py cli    # CLI interface

Edit file config.cfg sesuai kebutuhan

# Global configuration
FILE_EXCEL = _data/saku_data.xlsx
EXPORT_DIR = _exports

# Web server
HOST = 0.0.0.0
PORT = 8000
